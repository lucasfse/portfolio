Projet web

Artswipe

Artswipe est une application web conçue en équipe pour valoriser le patrimoine culturel français. L'utilisateur indique ses préférences en swipant des œuvres, puis l'application recommande des musées adaptés à ses goûts et à sa localisation.

Le projet a mobilisé un front-end React/Next.js, un back-end Node.js et une base PostgreSQL. L'algorithme combine les préférences de l'utilisateur avec les caractéristiques des œuvres et des musées, notamment les courants, les artistes et les catégories.

Next.js React TypeScript Node.js PostgreSQL Recommandation

Projet UQAC

AlgoViz

AlgoViz est une application pédagogique développée avec React et Vite pour rendre visibles des algorithmes souvent difficiles à suivre uniquement avec leur résultat final.

La plateforme propose plusieurs modules interactifs : Sudoku par backtracking, coloration de graphe avec DSATUR, plus court chemin avec Dijkstra et problème de tournées de véhicules. Chaque visualisation met en avant les choix de l'algorithme, les conflits, les mises à jour et la progression vers la solution.

React Vite React Router Dijkstra DSATUR Recherche taboue
